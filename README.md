# Calculadora-IMC
Trabalho de Desenvolvimento de FROND-END

1) Objetivo do projeto (resumido)
Criar uma página web que calcule o IMC, permitindo ao usuário inserir peso e altura, visualizar o resultado com a categoria correspondente e consultar um histórico das medições feitas durante a sessão.

2) Funcionalidades (resumido)
Inserção de peso (kg) e altura (cm ou m).
Botão Calcular IMC com validação dos dados.
Exibição do IMC, categoria e mensagem orientativa..
Layout responsivo, acessível e com visual simples e moderno.

3) Tecnologias (resumido)
HTML5
CSS3
JavaScript (ES6)
